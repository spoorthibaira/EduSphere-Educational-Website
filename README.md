# EduSphere-Educational-Website

EduSphere is a responsive educational website developed using **HTML5, CSS3, and JavaScript**. The website provides information about an educational institute, its courses, campus facilities, and allows students to register through an online contact form.

---

# 📌 Project Description

The purpose of this project is to design a modern educational website that is responsive, user-friendly, and visually attractive. It demonstrates the fundamentals of front-end web development using HTML, CSS, and JavaScript.

---

# 🚀 Features

- Responsive Navigation Bar
- Attractive Home Page with Hero Banner
- About Us Page
- Professional Courses Page
- Campus Gallery
- Student Registration Form
- JavaScript Form Validation
- Responsive Design for Mobile and Desktop
- Interactive Buttons and Hover Effects
- Clean and Modern UI

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
